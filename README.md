# Dashboard de Ventas - Desafío

## Objetivo del Proyecto
Desenvolver um dashboard de vendas organizado e interativo no Excel, com o propósito de facilitar a análise do desempenho de vendas e apoiar a tomada de decisões baseada em dados.


## Dados Utilizados
- Arquivo base: base.xlsx
- Colunas principais da base de dados:
  - Data da venda
  - Produto/Categoria de produto
  - Região de venda
  - Quantidade vendida
  - Valor total da venda


## Instruções para Reprodução
1. Faça o download dos arquivos base.xlsx (dados brutos) e dashboard_vendas_final.xlsx (dashboard concluído) do repositório.
2. Abra o arquivo dashboard_vendas_final.xlsx no Microsoft Excel.
3. Utilize os filtros dinâmicos disponíveis para segmentar os dados por período, região ou tipo de produto.
4. Analise os gráficos e métricas para visualizar o desempenho de vendas.


## Componentes do Dashboard
- Métricas chave: Venda total, quantidade total vendida e média de valor por venda.
- Gráfico de linhas: Evolução das vendas ao longo do tempo.
- Gráfico de barras: Vendas por região e por produto.
- Tabela resumo: Detalhamento de vendas por mês e categoria.
